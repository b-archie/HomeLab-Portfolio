# Active Directory - Home Lab Overview

This project demonstrates the deployment and administration of an Active Directory environment using VMware Workstation.

The lab includes:
- Windows Server 2022
- Domain Controller
- Active Directory Domain Services (AD:DS)
- Domain Name System (DNS)
- Dynamic Host Configuration Protocol (DHCP)
- Organizational Unit (OU) structure
- Group Policy Objects (GPO)
- Delegation of Control
- Domain-joined Windows 11 workstation
- VMware NAT networking
